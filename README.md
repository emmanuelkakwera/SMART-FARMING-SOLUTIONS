# Smart Farming Solutions 

A comprehensive web application providing agricultural guidance for soil management and animal husbandry practices, specifically tailored for Malawian farming conditions.

##  Features

### Soil Management
- **Soil Type Analysis**: Detailed information on different soil types found in Malawi
- **Crop Compatibility**: Guidance on which crops grow best in each soil type
- **Soil Activation**: Techniques to improve soil health and productivity
- **Comparison Tables**: Easy-to-understand soil comparisons

### Animal Husbandry
- **7 Animal Categories**: Chickens, Cattle, Goats, Pigs, Sheep, Ducks, and Rabbits
- **Breed Information**: Local and improved breeds with characteristics
- **Disease Management**: Comprehensive disease identification with images and treatments
- **Vaccination Schedules**: Complete vaccination timelines for each animal type
- **Management Practices**: Best practices for animal care and housing
